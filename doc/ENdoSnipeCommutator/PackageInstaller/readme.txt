##################################################
　ENdoSnipe Commutator
　必要ソフト自動インストールスクリプト
##################################################

・概要
　このスクリプトはENdoSnipe Commutatorの動作環境構築において
　インストール処理を自動的に行うスクリプト群です。

・
・利用方法
　/home/user_name/work/ にこのinstallerディレクトリを配置してください。
　ENdoSnipeCommutator環境構築手順書と同じディレクトリ構成になります。

　ダウンロードスクリプトで作成された、packagesディレクトリを
　そのままinstallerディレクトリに配置してください。

　シェルスクリプトを実行するとインストールが行われます。
