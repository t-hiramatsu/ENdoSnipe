##################################################
　ENdoSnipe Commutator
　必要ソフト自動ダウンロードスクリプト
##################################################

・概要
　このスクリプトはENdoSnipe Commutatorの動作環境構築において
　必要なファイルを自動的にダウンロードするスクリプトです。

・動作環境
  外部ネットワークに接続可能な環境でご利用ください。
  wgetコマンドが利用可能である必要があります。

・ファイル構成
  + list : 必要なファイルのURLが記載されたファイル
  + package : [スクリプト実行時に自動生成される]
              ダウンロードしたファイルが保存されるディレクトリ
  - readme.txt : このファイル
  - downloader.sh : ダウンロードスクリプト本体

・利用方法
　[パッケージをダウンロード]
　コマンドラインから以下の命令を実行してください。
　sh ./downloader.sh
